# Proposta Amélie Cookie Shop - Osasco Express

Pacote pronto para publicar no GitHub Pages.

## Importante para abrir corretamente
Suba o `index.html` e a pasta `assets` diretamente na raiz do repositório ou da pasta do GitHub Pages.

Estrutura correta:
- index.html
- assets/amelie-logo.png
- assets/amelie-fotos-produtos.png

Evite subir a pasta inteira `amelie_cookie_shop_proposta` dentro do repositório se o link esperado aponta para a raiz. Se subir a pasta fechada, o GitHub Pages pode não encontrar o `index.html` no endereço principal.
